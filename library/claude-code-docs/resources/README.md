## Changelog

### Recent Updates
Claude Code receives regular updates with new features, improvements, and bug fixes.

### Update Commands
```bash
# Check current version
claude --version

# Update to latest
claude update

# Install specific version
curl -fsSL https://claude.ai/install.sh | bash -s 1.0.58
```

### Release Channels
| Channel | Description |
|---------|-------------|
| `stable` | Production-ready releases (default) |
| `latest` | Most recent features |

Configure channel:
```json
{
  "releaseChannel": "stable"
}
```
